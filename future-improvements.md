# Future Improvements

## 1. Stabilita online

- Aggiungere heartbeat/ping per capire quali giocatori sono davvero connessi.
- Mostrare uno stato "riconnessione..." direttamente sui seat dei giocatori.
- Implementare host migration se l'host sparisce definitivamente.
- Rendere le lobby persistenti anche in caso di riavvio del server Render.

## 2. Esperienza mobile

- Aggiungere un bottone "Copia link" invece del solo link testuale.
- Migliorare il layout in landscape.
- Aggiungere vibrazione o feedback visivo quando e' il tuo turno.
- Prevenire zoom/input fastidiosi su iPhone.

## 3. UX di gioco

- Animare le carte giocate verso il tavolo.
- Animare la distribuzione delle carte.
- Rendere molto evidente l'indicatore "tocca a te".
- Migliorare il riepilogo mano con chiamata, prese e vite perse.

## 4. Regole e controlli

- Aggiungere una schermata impostazioni partita.
- Rendere configurabili numero vite, ciclo carte e varianti di regole.
- Aggiungere conferma prima di giocare la matta come zero o come alta.
- Gestire spettatori o giocatori eliminati con una UI dedicata.

## 5. Qualita tecnica

- Aggiungere test automatici sul motore Bisca.
- Aggiungere test sui casi multiplayer principali.
- Separare meglio UI, networking e game state.
- Salvare replay o log della partita per debug.

## 6. Look da card room

- Aggiungere avatar o iniziali nei seat.
- Aggiungere chip stack o monete decorative.
- Aggiungere suoni opzionali.
- Rendere l'animazione vittoria piu' scenografica.
- Aggiungere temi tavolo selezionabili.

## Priorita consigliata

Le prossime tre cose da fare:

1. Bottone "Copia link".
2. Indicatore di riconnessione sui seat.
3. Test automatici del motore di gioco.
